# WinDurango.github.io

Static project hub for [WinDurango](https://github.com/WinDurango/WinDurango), built with Vite, Svelte, TypeScript, SCSS, and Bun.

## Commands

```sh
bun install
bun run dev
bun run check
bun run build
bun run preview
```

The site is configured for static output and GitHub Pages deployment to `https://windurango.github.io/`.
